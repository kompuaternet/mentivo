# Mentivo Website

Static website for `https://www.mentivo.net/`.

## Pages

- `/` - Mentivo home
- `/support.html` - support page for all Mentivo apps
- `/apps/oracle-ball-ai/` - Oracle Ball AI product/support page
- `/legal/privacy.html` - Privacy Policy
- `/legal/terms.html` - Terms of Use
- `/legal/delete-data.html` - Delete Data instructions
- `/app-ads.txt` - AdMob publisher declaration

## Deploy

Upload the contents of this `Website` folder to the GitHub Pages repository that is connected to `www.mentivo.net`.

Keep `CNAME` in the root of the deployed site.

AdMob publisher declaration:

```text
google.com, pub-8843292959471328, DIRECT, f08c47fec0942fa0
```

Also make sure `support@mentivo.net` forwards to your real inbox.
